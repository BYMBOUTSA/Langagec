Le positionnement en CSS.

La position STATIC.

La valeur STATIC est la valeur par défaut lorsque l'on n'utilise pas la propriété POSITION d'un élément. Selon les spécifications de W3C, un élément HTML dont la propriété POSITION à la valeur "STATIC" est positioné dans le flux normal. Cela signifie qu'un élément statique sera positionné à la suite des autres éléments, le changement de son comportement dépendant de sa nature block ou inline. Il ne pourra en aucun cas sortir du flux global de son conteneur.
